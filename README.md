# 🏥 WhatsApp AI Enquiry Handler (Al-Shifa Diagnostic Center)

An intelligent, AI-powered patient support agent designed to automate and streamline WhatsApp enquiries for diagnostic centers. Built with **Google Gemini 1.5 Flash** and **Streamlit**.

---

## 🚀 Key Features

* **Intelligent Classification:** Automatically categorizes patient messages into 5 distinct categories (Appointments, Pricing, Reports, Home Healthcare, or Other).
* **Context-Aware Responses:** Generates professional, warm, and concise replies based on the enquiry type.
* **Smart Human Handoff:** Instantly flags urgent or sensitive messages (e.g., "emergency", "complaint") for manual staff intervention.
* **Automated Logging:** Maintains a local `enquiry_log.json` file with timestamps and AI responses for audit trails.
* **Real-time Dashboard:** A clean Streamlit interface to simulate patient interactions.

---

## 🛠️ Tech Stack

* **Language:** Python
* **AI Model:** Google Gemini 2.5 Flash
* **Frontend:** Streamlit
